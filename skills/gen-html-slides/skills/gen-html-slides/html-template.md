# HTML Presentation Template

Reference architecture for generating slide presentations. Every presentation follows this structure.

## Base HTML Structure

```html
<!DOCTYPE html>
<html lang="en">
  <head>
    <meta charset="UTF-8" />
    <meta name="viewport" content="width=device-width, initial-scale=1.0" />
    <title>Presentation Title</title>

    <!-- Fonts: use Fontshare or Google Fonts — never system fonts -->
    <link rel="stylesheet" href="https://api.fontshare.com/v2/css?f[]=..." />

    <style>
      /* ===========================================
           CSS CUSTOM PROPERTIES (THEME)
           Change these to change the whole look
           =========================================== */
      :root {
        /* Colors — from chosen style preset */

        --text-primary: #ffffff;

        --slide-padding: clamp(1.5rem, 4vw, 4rem);

      <div class="slide-content">
        <h2 class="reveal">Slide Title</h2>
```

## Required JavaScript Features

Every presentation must include:

1. **SlidePresentation Class** — Main controller with:
   - Keyboard navigation (arrows, space, page up/down)
   - Touch/swipe support
   - Mouse wheel navigation
   - Progress bar updates
   - Navigation dots

2. **Intersection Observer** — For scroll-triggered animations:
   - Add `.visible` class when slides enter viewport
   - Trigger CSS transitions efficiently

3. **Optional Enhancements** (match to chosen style):
   - Custom cursor with trail
   - Particle system background (canvas)
   - Parallax effects
   - 3D tilt on hover
   - Magnetic buttons
   - Counter animations

4. **Inline Editing** (only if user opted in during Phase 1 — skip entirely if they said No):
   - Edit toggle button (hidden by default, revealed via hover hotzone or `E` key)
   - Auto-save to localStorage
   - Export/save file functionality
   - See "Inline Editing Implementation" section below

## Inline Editing Implementation (Opt-In Only)

**If the user chose "No" for inline editing in Phase 1, do NOT generate any edit-related HTML, CSS, or JS.**

**IMPORTANT — Language consistency:** All Inline Editor UI strings (button label, banner text, keyboard hint in `title` attribute) **must match the presentation's language**. The templates below use English defaults; a translation table is provided at the end of this section.

**Do NOT use CSS `~` sibling selector for hover-based show/hide.** The CSS-only approach (`edit-hotzone:hover ~ .edit-toggle`) fails because `pointer-events: none` on the toggle button breaks the hover chain: user hovers hotzone -> button becomes visible -> mouse moves toward button -> leaves hotzone -> button disappears before click.

**Required approach: JS-based hover with 400ms delay timeout.**

HTML:

```html
<div class="edit-hotzone"></div>
<button class="edit-toggle" id="editToggle" title="Edit mode (E)">✏️</button>
```

CSS (visibility controlled by JS classes only):

```css
/* Do NOT use CSS ~ sibling selector for this!
   pointer-events: none breaks the hover chain.
   Must use JS with delay timeout. */
.edit-hotzone {
  position: fixed;
  top: 0;
  left: 0;
  width: 80px;
  height: 80px;
  z-index: 10000;
  cursor: pointer;
}
.edit-toggle {
  opacity: 0;
  pointer-events: none;
  transition: opacity 0.3s ease;
  z-index: 10001;
}
.edit-toggle.show,
.edit-toggle.active {
  opacity: 1;
  pointer-events: auto;
}
```

JS (three interaction methods):

```javascript
// 1. Click handler on the toggle button
document.getElementById("editToggle").addEventListener("click", () => {
  editor.toggleEditMode();
});

// 2. Hotzone hover with 400ms grace period
const hotzone = document.querySelector(".edit-hotzone");
const editToggle = document.getElementById("editToggle");
let hideTimeout = null;

hotzone.addEventListener("mouseenter", () => {
  clearTimeout(hideTimeout);
  editToggle.classList.add("show");
});
hotzone.addEventListener("mouseleave", () => {
  hideTimeout = setTimeout(() => {
    if (!editor.isActive) editToggle.classList.remove("show");
  }, 400);
});
editToggle.addEventListener("mouseenter", () => {
  clearTimeout(hideTimeout);
});
editToggle.addEventListener("mouseleave", () => {
  hideTimeout = setTimeout(() => {
    if (!editor.isActive) editToggle.classList.remove("show");
  }, 400);
});

// 3. Hotzone direct click
hotzone.addEventListener("click", () => {
  editor.toggleEditMode();
});

// 4. Keyboard shortcut (E key, skip when editing text)
document.addEventListener("keydown", (e) => {
  if (
    (e.key === "e" || e.key === "E") &&
    !e.target.getAttribute("contenteditable")
  ) {
    editor.toggleEditMode();
  }
});
```

**CRITICAL: `exportFile()` must strip edit state before capturing outerHTML.**

When the user presses Ctrl+S in edit mode, `document.documentElement.outerHTML` captures the live DOM —
including `body.edit-active`, `contenteditable="true"` on every text element, and `.active`/`.show` classes on
the toggle button and banner. Anyone opening the saved file sees dashed outlines, a checkmark button, and an
edit banner, as if permanently stuck in edit mode.

Always implement `exportFile()` like this:

```javascript
exportFile() {
    // Temporarily strip edit state so the saved file opens cleanly
    const editableEls = Array.from(document.querySelectorAll('[contenteditable]'));
    editableEls.forEach(el => el.removeAttribute('contenteditable'));
    document.body.classList.remove('edit-active');

    // Also strip UI classes from toggle button and banner
    const editToggle = document.getElementById('editToggle');
    const editBanner = document.querySelector('.edit-banner');
    editToggle?.classList.remove('active', 'show');
    editBanner?.classList.remove('active', 'show');

    const html = '<!DOCTYPE html>\n' + document.documentElement.outerHTML;

    // Restore edit state so the user can keep editing
    document.body.classList.add('edit-active');
    editableEls.forEach(el => el.setAttribute('contenteditable', 'true'));
    editToggle?.classList.add('active');
    editBanner?.classList.add('active');

    const blob = new Blob([html], { type: 'text/html' });
    const a = document.createElement('a');
    a.href = URL.createObjectURL(blob);
    a.download = 'presentation.html';
    a.click();
    URL.revokeObjectURL(a.href);
}
```

### Inline Editor — Localizable UI Strings

When implementing `toggleEditMode()`, use these strings — **localized to the presentation's language**:

**English (default):**
```javascript
// Button title attribute (HTML)
title="Edit mode (E)"

// toggleEditMode() — entering edit mode
editToggle.textContent = '✏️ Done';          // button label while active
editBanner.textContent = '✏️ Edit Mode — click any text · Ctrl+S to save';

// toggleEditMode() — leaving edit mode
editToggle.textContent = '✏️ Edit';          // button label while inactive
```

**Chinese equivalents** (use when the presentation language is Chinese):
```javascript
title="编辑模式 (E)"

editToggle.textContent = '✏️ 完成';
editBanner.textContent = '✏️ 编辑模式 — 点击文字编辑 · Ctrl+S 保存';

editToggle.textContent = '✏️ 编辑';
```

**Rule:** Detect the presentation language from `<html lang="...">` or from the content language, and apply the matching set of strings above.

## Image Manager (Opt-In Only)

**If the user chose "No" for image manager in Phase 1, do NOT generate any image manager HTML, CSS, or JS.**

This feature lets users upload and place images directly in the browser — no HTML editing required. Images are base64-encoded via FileReader, keeping the single HTML file fully portable.

Two modes are available through a two-tab panel:
- **Add Image tab** — upload an image, choose freeform overlay or merge into a structured layout slot
- **Slide Layout tab** — apply a layout preset to the slide (text+image split), upload a slot image, and drag the divider to adjust proportions

**IMPORTANT — Language consistency:** All Image Manager UI strings (tab labels, button text, placeholders, PRESETS labels, size names) **must match the presentation's language**. If the presentation is in English, use English. If Chinese, use Chinese. The templates below use English; a translation table is provided at the end of this section.

**If BOTH inline editing AND image manager are enabled:** update `exportFile()` to deselect managed images before capturing outerHTML:

```javascript
// At the top of exportFile(), before stripping edit state:
document.querySelectorAll('.managed-img-wrap.selected').forEach(w => w.classList.remove('selected'));
```

### Slide HTML Structure (required)

Every `<section class="slide ...">` **must** wrap its content in `.slide-text-wrap` and include an empty `.slide-img-slot` after it. The slot starts hidden; layout preset classes reveal and size it.

```html
<section class="slide slide-title" id="slide-0">
  <div class="slide-text-wrap">
    <div class="slide-content">
      <!-- all slide content here (headings, lists, etc.) -->
    </div>
  </div>
  <div class="slide-img-slot"></div>
</section>
```

```javascript
class ImageManager {
  constructor(presentation) {
    this.presentation = presentation;  // SlidePresentation instance for current-slide tracking
    this.freeformDataUrl = null;
    this.slotDataUrl = null;
    this.freeformLayout = 'auto';
    this.selectedLayoutPreset = null;

    // Labels use the presentation's language (see translation table at end of section)

      document.querySelectorAll('#presetGrid .preset-btn').forEach(b => b.classList.remove('active'));

  _fillGrid(grid, presets, onSelect, defaultId) {
    presets.forEach(p => {
      const btn = document.createElement('button');
      const isDefault = p.id === defaultId || (defaultId === null && p.id === null);
      btn.className = 'preset-btn' + (isDefault ? ' active' : '');

      ...t.texts.map(r => `<rect x="${r.x}" y="${r.y}" width="${r.w}" height="${r.h}" rx="1" fill="rgba(255,255,255,0.3)"/>`),

  _loadFile(file, target) {

  _openPanel() {

  _closePanel() {

  _populateSlideSelects() {
    const slides = document.querySelectorAll('.slide');

    const currentIdx = this.presentation ? this.presentation.currentSlide : 0;

    ['imgSlideSelect', 'layoutSlideSelect'].forEach(id => {

      slides.forEach((slide, i) => {

    const sel = document.getElementById('layoutSlideSelect');
    if (sel._syncHandler) sel.removeEventListener('change', sel._syncHandler);
    sel._syncHandler = e => this._syncPresetBtns(parseInt(e.target.value));

  _syncPresetBtns(slideIdx) {
    const slide = document.querySelectorAll('.slide')[slideIdx];
    const activePreset = this.PRESETS.find(p => p.id && slide.classList.contains(p.id));
    document.querySelectorAll('.preset-btn').forEach(btn => {
      btn.classList.toggle('active', btn.dataset.preset === (activePreset?.id || 'default'));
    });
    this.selectedLayoutPreset = activePreset?.id || null;
  }

  /* ── FREE-FORM IMAGE PLACEMENT ── */
  _addFreeformImage() {
    if (!this.freeformDataUrl) return;
    const slideIdx = parseInt(document.getElementById('imgSlideSelect').value);
    const slide = document.querySelectorAll('.slide')[slideIdx];

    if (this.addMode === 'merge') {
      this._applyLayoutWithImage(slide, this.addMergePreset, this.freeformDataUrl);
    } else {
      const sizePct = parseInt(document.getElementById('imgSizeRange').value);
      const pos = this._calcFreeformPos(slide, this.freeformLayout, sizePct);
      const wrap = this._makeFreeformWrap(this.freeformDataUrl, pos);
      slide.appendChild(wrap);
      this._bindDragResize(wrap, slide);
    }
    this._closePanel();
  }

  _applyLayoutWithImage(slide, preset, imageSrc) {
    const slot = slide.querySelector('.slide-img-slot');
    const textWrap = slide.querySelector('.slide-text-wrap');
    ['lyt-ti','lyt-it','lyt-tb','lyt-bt','lyt-full'].forEach(c => slide.classList.remove(c));
    slide.querySelector('.layout-divider')?.remove();
    if (textWrap.nextElementSibling !== slot) slide.appendChild(slot);
    if (!preset) return;
    if (preset === 'lyt-it' || preset === 'lyt-bt') slide.insertBefore(slot, textWrap);
    slide.classList.add(preset);
    slide.querySelector('.managed-img-wrap')?.remove();
    this._setSlotImage(slot, imageSrc);
    if (preset !== 'lyt-full') this._addDivider(slide, slot, preset);
  }

  _makeFreeformWrap(src, pos) {
    const wrap = document.createElement('div');
    wrap.className = 'managed-img-wrap';
    wrap.style.cssText = `left:${pos.left};top:${pos.top};width:${pos.width};height:${pos.height}`;
    const img = document.createElement('img'); img.src = src; img.alt = ''; img.draggable = false;
    const del = document.createElement('button'); del.className = 'mimg-del'; del.innerHTML = '×'; del.title = '删除';
    const rsz = document.createElement('div'); rsz.className = 'mimg-resize';
    del.addEventListener('click', e => { e.stopPropagation(); wrap.remove(); });
    wrap.append(img, del, rsz);
    wrap.addEventListener('click', e => { e.stopPropagation(); this._selectFreeform(wrap); });
    return wrap;
  }

  _calcFreeformPos(slide, layout, sizePct) {
    const w = sizePct, h = Math.round(w * 0.62);
    const map = {
      right:      { left:`${97-w}%`,      top:`${(100-h)/2}%`, width:`${w}%`, height:`${h}%` },
      left:       { left:'3%',            top:`${(100-h)/2}%`, width:`${w}%`, height:`${h}%` },
      bottom:     { left:`${(100-w)/2}%`, top:`${96-h}%`,      width:`${w}%`, height:`${h}%` },
      center:     { left:`${(100-w)/2}%`, top:`${(100-h)/2}%`, width:`${w}%`, height:`${h}%` },
      background: { left:'0%', top:'0%', width:'100%', height:'100%' },
    };
    if (layout !== 'auto') return map[layout] || map.center;
    const hasBody = slide.querySelector('p,ul,ol,[class*="content"]');
    const hasHead = slide.querySelector('h1,h2');
    if (hasBody) return map.right;
    if (hasHead) return { ...map.center, top: '42%' };
    return map.center;
  }

  _selectFreeform(wrap) {
    document.querySelectorAll('.managed-img-wrap.selected').forEach(w => w.classList.remove('selected'));
    wrap.classList.add('selected');
  }

  _bindDragResize(wrap, slide) {
    let dragging = false, resizing = false, sx, sy, sl, st, sw, sh;
    const del = wrap.querySelector('.mimg-del');
    const rsz = wrap.querySelector('.mimg-resize');

    wrap.addEventListener('mousedown', e => {
      if (!wrap.classList.contains('selected') || e.target === rsz || e.target === del) return;
      dragging = true; sx = e.clientX; sy = e.clientY; sl = wrap.offsetLeft; st = wrap.offsetTop; e.preventDefault();
    });
    rsz.addEventListener('mousedown', e => {
      resizing = true; sx = e.clientX; sy = e.clientY; sw = wrap.offsetWidth; sh = wrap.offsetHeight;
      e.preventDefault(); e.stopPropagation();
    });
    document.addEventListener('mousemove', e => {
      if (dragging) {
        const ww = slide.offsetWidth, wh = slide.offsetHeight;
        const nl = Math.max(0, Math.min(ww - wrap.offsetWidth,  sl + (e.clientX - sx)));
        const nt = Math.max(0, Math.min(wh - wrap.offsetHeight, st + (e.clientY - sy)));
        wrap.style.left = (nl/ww*100).toFixed(2)+'%'; wrap.style.top = (nt/wh*100).toFixed(2)+'%';
      }
      if (resizing) {
        const ww = slide.offsetWidth, wh = slide.offsetHeight;
        wrap.style.width  = (Math.max(40, sw + (e.clientX - sx)) / ww * 100).toFixed(2)+'%';
        wrap.style.height = (Math.max(30, sh + (e.clientY - sy)) / wh * 100).toFixed(2)+'%';
      }
    });
    document.addEventListener('mouseup', () => { dragging = false; resizing = false; });
  }

  /* ── LAYOUT PRESET APPLICATION ── */
  _applyLayout() {
    const slideIdx = parseInt(document.getElementById('layoutSlideSelect').value);
    const slide = document.querySelectorAll('.slide')[slideIdx];
    const slot = slide.querySelector('.slide-img-slot');
    const textWrap = slide.querySelector('.slide-text-wrap');

    ['lyt-ti','lyt-it','lyt-tb','lyt-bt','lyt-full'].forEach(c => slide.classList.remove(c));
    slide.querySelector('.layout-divider')?.remove();
    if (textWrap.nextElementSibling !== slot) slide.appendChild(slot);

    const preset = this.selectedLayoutPreset;

    if (!preset) {
      // Revert to text-only: migrate slot image to freeform if present
      const slotImg = slot.querySelector('.slot-image');
      if (slotImg) this._slotToFreeform(slide, slot, slotImg.src);
      slot.innerHTML = '';
      this._closePanel(); return;
    }

    // lyt-it and lyt-bt need the slot physically before textWrap in DOM
    // so _addDivider can always insert between them using compareDocumentPosition
    if (preset === 'lyt-it' || preset === 'lyt-bt') slide.insertBefore(slot, textWrap);

    slide.classList.add(preset);

    const imageSrc = this.slotDataUrl
      || slot.querySelector('.slot-image')?.src
      || slide.querySelector('.managed-img-wrap img')?.src;

    if (imageSrc) {
      if (this.slotDataUrl || !slot.querySelector('.slot-image')) {
        const ff = slide.querySelector('.managed-img-wrap');
        if (ff && !this.slotDataUrl) ff.remove();
      }
      this._setSlotImage(slot, imageSrc);
    } else {
      this._setSlotPlaceholder(slot);
    }

    this._addDivider(slide, slot, preset);
    this.slotDataUrl = null;
    this._closePanel();
  }

  _addDivider(slide, slot, preset) {
    slide.querySelector('.layout-divider')?.remove();
    if (!preset || preset === 'lyt-full') return;

    const divider = document.createElement('div');
    divider.className = 'layout-divider';

    // Insert divider between text-wrap and slot regardless of DOM order.
    // compareDocumentPosition avoids the slot.nextElementSibling === null trap
    // that arises when textWrap comes first (lyt-ti / lyt-tb).
    const textWrap = slide.querySelector('.slide-text-wrap');
    const slotIsAfterText = !!(textWrap.compareDocumentPosition(slot) & Node.DOCUMENT_POSITION_FOLLOWING);
    slide.insertBefore(divider, slotIsAfterText ? slot : textWrap);

    const isHorizontal = preset === 'lyt-ti' || preset === 'lyt-it';
    this._bindDividerDrag(divider, slide, slot, isHorizontal, preset);
  }

  _bindDividerDrag(divider, slide, slot, isHorizontal, preset) {
    let dragging = false;

    const onDown = e => { dragging = true; divider.classList.add('dragging'); e.preventDefault(); e.stopPropagation(); };
    const onMove = e => {
      if (!dragging) return;
      const rect = slide.getBoundingClientRect();
      let pct;
      if (isHorizontal) {
        const mouseX = (e.clientX ?? e.touches?.[0]?.clientX) - rect.left;
        // lyt-ti: slot on RIGHT → pct grows as mouse moves left
        // lyt-it: slot on LEFT  → pct grows as mouse moves right
        pct = preset === 'lyt-ti'
          ? (rect.width - mouseX) / rect.width * 100
          : mouseX / rect.width * 100;
      } else {
        const mouseY = (e.clientY ?? e.touches?.[0]?.clientY) - rect.top;
        // lyt-tb: slot on BOTTOM → pct grows as mouse moves up
        // lyt-bt: slot on TOP    → pct grows as mouse moves down
        pct = preset === 'lyt-tb'
          ? (rect.height - mouseY) / rect.height * 100
          : mouseY / rect.height * 100;
      }
      pct = Math.max(10, Math.min(80, pct));
      slot.style.flex = `0 0 ${pct.toFixed(1)}%`;
    };
    const onUp = () => { if (dragging) { dragging = false; divider.classList.remove('dragging'); } };

    divider.addEventListener('mousedown', onDown);
    divider.addEventListener('touchstart', onDown, { passive: false });
    document.addEventListener('mousemove', onMove);
    document.addEventListener('touchmove', onMove, { passive: false });
    document.addEventListener('mouseup', onUp);
    document.addEventListener('touchend', onUp);
  }

  _setSlotImage(slot, src) {
    slot.innerHTML = '';
    const wrap = document.createElement('div');
    wrap.className = 'slot-image-wrap';
    wrap.style.cssText = 'width:100%;height:100%;position:relative';
    const img = document.createElement('img');
    img.src = src; img.className = 'slot-image'; img.alt = '';
    const replaceBtn = document.createElement('button');
    replaceBtn.className = 'slot-replace-btn'; replaceBtn.innerHTML = '×'; replaceBtn.title = '移除图片';
    replaceBtn.addEventListener('click', e => {
      e.stopPropagation();
      this._setSlotPlaceholder(slot);
      const s = slot.closest('.slide');
      ['lyt-ti','lyt-it','lyt-tb','lyt-bt','lyt-full'].forEach(c => s.classList.remove(c));
      s.querySelector('.layout-divider')?.remove();
      const tw = s.querySelector('.slide-text-wrap');
      if (tw && tw.nextElementSibling !== slot) s.appendChild(slot);
    });
    wrap.append(img, replaceBtn);
    slot.appendChild(wrap);
  }

  _setSlotPlaceholder(slot) {
    slot.innerHTML = '';
    const ph = document.createElement('div'); ph.className = 'img-slot-placeholder';
    ph.innerHTML = '<span class="slot-icon">📷</span><span>Click to upload layout image</span>';
    ph.addEventListener('click', () => {
      document.querySelectorAll('.panel-tab').forEach(t => t.classList.remove('active'));
      document.querySelectorAll('.panel-pane').forEach(p => p.classList.remove('active'));
      document.querySelector('[data-tab="layout"]').classList.add('active');
      document.getElementById('pane-layout').classList.add('active');
      this._openPanel();
    });
    slot.appendChild(ph);
  }

  _slotToFreeform(slide, slot, src) {
    const pos = this._calcFreeformPos(slide, 'center', 40);
    const wrap = this._makeFreeformWrap(src, pos);
    slide.appendChild(wrap);
    this._bindDragResize(wrap, slide);
  }

  /* ── DESELECT & KEYBOARD ── */
  _bindClickDeselect() {
    document.addEventListener('click', e => {
      if (!e.target.closest('.managed-img-wrap') && !e.target.closest('#imgMgrOverlay')) {
        document.querySelectorAll('.managed-img-wrap.selected').forEach(w => w.classList.remove('selected'));
      }
    });
    document.addEventListener('keydown', e => {
      if (e.key === 'Escape') {
        document.querySelectorAll('.managed-img-wrap.selected').forEach(w => w.classList.remove('selected'));
        this._closePanel();
      }
      if ((e.key === 'Delete' || e.key === 'Backspace') && !e.target.getAttribute('contenteditable')) {
        document.querySelectorAll('.managed-img-wrap.selected').forEach(w => w.remove());
      }
    });
  }

  _bindKeyboard() {
    document.addEventListener('keydown', e => {
      if ((e.key === 'i' || e.key === 'I') && !e.target.getAttribute('contenteditable') && !e.metaKey && !e.ctrlKey) {
        document.getElementById('imgMgrOverlay').classList.contains('active')
          ? this._closePanel() : this._openPanel();
      }
    });
  }

  /* ── RE-ATTACH (saved/reopened files) ── */
  _reattachExisting() {
    // Re-bind freeform images already in the DOM
    document.querySelectorAll('.managed-img-wrap').forEach(wrap => {
      const slide = wrap.closest('.slide');
      if (!slide) return;
      const rsz = wrap.querySelector('.mimg-resize');
      const del = wrap.querySelector('.mimg-del');
      if (rsz) this._bindDragResize(wrap, slide);
      if (del) del.addEventListener('click', e => { e.stopPropagation(); wrap.remove(); });
      wrap.addEventListener('click', e => { e.stopPropagation(); this._selectFreeform(wrap); });
    });
    // Re-bind layout dividers already in the DOM
    const LAYOUT_CLASSES = ['lyt-ti','lyt-it','lyt-tb','lyt-bt'];
    document.querySelectorAll('.slide').forEach(slide => {
      const preset = LAYOUT_CLASSES.find(c => slide.classList.contains(c));
      if (!preset) return;
      const divider = slide.querySelector('.layout-divider');
      const slot = slide.querySelector('.slide-img-slot');
      if (divider && slot) {
        const isHorizontal = preset === 'lyt-ti' || preset === 'lyt-it';
        this._bindDividerDrag(divider, slide, slot, isHorizontal, preset);
      }
    });
  }
}
```

### Image Processing

```python
from PIL import Image, ImageDraw

# Circular crop (for logos on modern/clean styles)
def crop_circle(input_path, output_path):
    img = Image.open(input_path).convert('RGBA')
    w, h = img.size
    size = min(w, h)
    left, top = (w - size) // 2, (h - size) // 2
    img = img.crop((left, top, left + size, top + size))
    mask = Image.new('L', (size, size), 0)
    ImageDraw.Draw(mask).ellipse([0, 0, size, size], fill=255)
    img.putalpha(mask)
    img.save(output_path, 'PNG')

# Resize (for oversized images that inflate HTML)
def resize_max(input_path, output_path, max_dim=1200):
    img = Image.open(input_path)
    img.thumbnail((max_dim, max_dim), Image.LANCZOS)
    img.save(output_path, quality=85)
```

| Situation                        | Operation                     |
| -------------------------------- | ----------------------------- |
| Square logo on rounded aesthetic | `crop_circle()`               |
| Image > 1MB                      | `resize_max(max_dim=1200)`    |
| Wrong aspect ratio               | Manual crop with `img.crop()` |

Save processed images with `_processed` suffix. Never overwrite originals.
